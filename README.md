A professional one-page portfolio website built using **HTML5**, **CSS3**, **LaTeX**, and **GitHub Pages**. This project showcases my academic background, technical skills, projects, and downloadable CV while demonstrating modern web development practices and version control using Git.

This portfolio was developed as part of an academic project to create a professional online presence suitable for internships, working student positions, and software engineering opportunities.

The website emphasizes:

* Clean and semantic HTML structure
* Modular CSS styling

## Deployment

The project is deployed using **GitHub Pages**.

Deployment steps:

1. Push the project to GitHub.
2. Open **Repository Settings**.
3. Navigate to **Pages**.
4. Select the **main** branch.
5. Choose the **root (/)** folder.
6. Save the settings.

GitHub automatically publishes the website.
